致使用者

在这个一切都被要求云端同步、算法推荐、社交分享的时代，流萤选择了一条逆行的路。我们不收集你的阅读偏好，不分析你的点击热力，甚至不需要你注册一个账户。你的书库安静地躺在本地 IndexedDB 或 SQLite 的深处，如同藏在阁楼里的旧书箱，积着时间的薄灰，只属于你。

我们相信，阅读是一场私密的对话。那些在纸页边缘留下的折痕，那些深夜独自与主人公共度的时刻，那些因某段文字而突然停顿的呼吸——这些都不需要被上传至任何地方。这里是你的私人图书馆，五十本书的容量，足够容纳一个灵魂在某个季节的全部漫游。

阅读应当是沉浸的，如同潜入深水；
而非嘈杂的，如同在广场上被推销。
静默的技艺

本地永生

支持 IndexedDB 与 SQLite 双引擎存储。你的藏书永不上云，支持 JSON 批量导出备份，为你的私人图书馆制作副本。

声临其境

集成 Edge 语音合成，支持句子级高亮与选区朗读。让文字化为声音，在通勤路上或睡前，继续那场未完的阅读。

智能脉络

自动识别章节层级，支持自定义正则匹配。无论是"第一章"的古典还是"1.1.1"的精密，目录如植物脉络般展开。

全文寻踪

即时全文搜索，毫秒级定位关键词所在。支持高亮跳转，在数十万字的卷帙中，迅速找到那个名字或句子。

格式自由

支持 DOCX、TXT、Markdown、HTML 导入。导出支持 DOCX（B5 精致排版）与带目录的 HTML，让你的批注与排版随身携带。

四重光境

浅色如晨间窗前，复古似陈年羊皮纸，护眼像午后竹林，深色若深夜案头。四种配色精心调校，确保长时间阅读的舒适。

隐私誓言

流萤阅读器完全离线运行。你的文件不会被发送到任何服务器，你的阅读进度、章节设置、甚至搜索历史都只存在于你的设备本地。这是一个完全发生在浏览器沙盒内的应用，断网亦可使用。

由于数据完全本地存储，若你清除浏览器缓存，书库将随之消散。请定期使用书库管理中的批量导出功能，为你的藏书制作 JSON 备份。这像是为自己的私人图书馆制作一份副本，藏在另一个安全的角落。
