# 致初次见面的你

在这个一切都被要求云端同步、算法推荐、社交分享的时代，流萤选择了一条逆行的路。

我们不收集你的阅读偏好，不分析你的点击热力，甚至不需要你注册一个账户。你的书库安静地躺在本地存储的深处，如同藏在阁楼里的旧书箱，积着时间的薄灰，只属于你。

这本小册子将指引你熟悉流萤的每一处角落。愿你在这些纸页间，找到属于自己的流动的光。

> 纸页之间的私密光晕，只为相信文字应当被温柔对待的人。

## 快速开始

### 打开一本书

点击界面左上角的**文件夹图标**，或是中央醒目的**开始阅读按钮**，即可从设备中选择文档。流萤支持以下格式：

- **DOCX** —— 保留原有的章节层级与图片
- **TXT** —— 自动识别 UTF-8、GBK 等编码，无需担心乱码
- **Markdown** —— 支持代码高亮与富文本渲染
- **HTML** —— 完整的网页文档导入
- **PDF** —— 支持文字提取与图片展示（部分扫描版 PDF 可能无法识别文字）

### 从上次中断处继续

流萤会自动记录你的阅读进度。当你重新打开一本书，它会贴心地回到你上次离开的地方。

## 界面导览

### 顶部工具栏

如同书桌上的一排文具，顶部工具栏放置了最常用的功能：

- **目录图标** —— 展开左侧章节导航，像翻阅一本书的索引
- **文件夹图标** —— 打开新书
- **搜索图标** —— 在整本书中查找关键词
- **声音图标** —— 开启语音朗读，让文字化为声音
- **书签图标** —— 查看你的批注与收藏
- **时钟图标** —— 打开阅读历史与书库管理
- **齿轮图标** —— 进入阅读设置

### 左侧目录栏

这里是整本书的脉络。流萤会自动识别章节标题，为你生成清晰的层级结构。点击任意章节，即可瞬间跳转。

若你的文档使用了特殊的章节标记（如"卷一·第一章"而非标准的"第一章"），可在设置中自定义正则表达式来匹配。

### 右侧设置面板

在这里，你可以将阅读器调校为最舒适的状态：

- **语言** —— 简体中文、繁体中文、英文
- **主题** —— 二十种精心调配的配色，从晨光到夜幕
- **字体** —— 宋体的典雅、黑体的简洁、楷书的温润
- **排版** —— 字号、行距、段距、页边距的精细调节
- **章节编号** —— 中文、英文、罗马数字、学术格式任选

## 核心功能

### 语音朗读

点击顶部工具栏的**声音图标**，即可开启 TTS 语音朗读。流萤会逐句高亮当前朗读的内容，让你视听同步。

在设置面板中，你可以调节：
- **语速** —— 从从容不迫到一目十行
- **音调** —— 选择最悦耳的声音

### 智能章节识别

流萤内置了强大的章节识别引擎，能够自动识别以下格式：

- 第一章 / 第二章 / 第三章
- Chapter 1 / Chapter 2
- 1. 绪论 / 1.1 研究背景 / 1.1.1 具体问题
- （一）/（二）/（三）

若你的文档使用了特殊的章节标记，可在设置的**正则表达式**区域自定义匹配规则。流萤还提供了便捷的宏系统：

- `\C` —— 匹配中文数字（一二三四...）
- `\R` —— 匹配大写罗马数字（IVXLCDM）
- `\N` —— 匹配阿拉伯数字
- `\W` —— 匹配英文单词

### 全文搜索

点击**搜索图标**或按下键盘的 `F` 键，即可打开搜索面板。输入关键词，流萤会在整本书中 instant 检索，毫秒级呈现结果。

每个结果都会显示上下文片段，点击即可跳转至该处，并用温暖的橙色高亮标记。

### 书签与批注

阅读时，选中任意文字，会浮现一个小菜单。你可以：

- **添加批注** —— 写下此刻的感悟
- **添加书签** —— 标记这个位置，日后再来

点击顶部工具栏的**书签图标**，即可查看所有的批注与收藏。

### 热力图

在设置中，你可以定义一组**G点关键词**（如主角的名字、核心概念等）。流萤会在目录中以颜色的深浅，标示这些关键词在各章节的分布密度——哪里是情节的高潮，哪里是知识的密集区，一目了然。

## 键盘密语

对于偏爱键盘的读者，这些快捷键如同书页间的秘道：

| 快捷键 | 功能 |
|--------|------|
| `↑` `↓` | 上一章 / 下一章 |
| `←` `→` | 向上翻屏 / 向下翻屏 |
| `F` | 打开全文搜索 |
| `R` | 开启 / 关闭语音朗读 |
| `H` | 打开阅读历史 |
| `B` | 打开书库管理 |
| `S` | 打开设置面板 |
| `M` | 在当前位置添加书签 |
| `Esc` | 关闭当前面板 |

移动端支持左右滑动切换章节，双指缩放调整字号。

## 书库管理

### 本地存储模式

流萤默认使用浏览器的 IndexedDB 存储你的书籍。这是一种完全本地的存储方式：

- **容量** —— 约可存放 50 本书（视单本书大小而定）
- **特点** —— 完全离线，无需网络，隐私性最佳
- **注意** —— 清除浏览器缓存会导致书库消散，请定期备份

### 服务器存储模式

如果你在电脑上使用流萤的 Web 版本，可以启动附带的 Python 服务器，启用 SQLite 存储模式：

- **容量** —— 理论上无上限
- **特点** —— 配合本地缓存，打开速度极快
- **注意** —— 需要保持服务器运行

### 导入与导出

点击书库管理页面的**导出按钮**，可以将你的藏书打包为 JSON 文件。这像是为自己的私人图书馆制作一份副本，藏在另一个安全的角落。

**导入**功能则允许你恢复这份备份，或是将多本书一次性迁入流萤。

## 隐私与安全

流萤阅读器完全离线运行。你的文件不会被发送到任何服务器，你的阅读进度、章节设置、甚至搜索历史都只存在于你的设备本地。

这是一个完全发生在浏览器沙盒内的应用，断网亦可使用。

由于数据完全本地存储，若你清除浏览器缓存，书库将随之消散。请定期使用书库管理中的**批量导出**功能，为你的藏书制作 JSON 备份。

## 常见问题

**Q：为什么有些 PDF 打开后没有文字？**
A：流萤只能提取文本型 PDF 的内容。扫描版 PDF（图片形式）需要通过 OCR 软件先转换为可识别文本。

**Q：书库里最多能放多少本书？**
A：浏览器存储模式下约 50 本，服务器模式下无限制。建议定期整理，让书库保持轻盈。

**Q：语音朗读没有声音？**
A：请检查系统音量与浏览器标签页是否被静音。部分浏览器可能需要用户先与页面交互（点击任意位置）才允许播放声音。

**Q：章节识别不准确怎么办？**
A：打开设置面板，在**正则表达式**区域输入你的章节匹配规则。例如，若章节标记为"卷一·第一章"，可尝试模式 `^卷\C·第\C章`。
