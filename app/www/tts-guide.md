# 语音朗读：让文字化为声音

阅读并非一定要用眼睛。

当你想要解放双手，或是疲惫了一天后只想闭眼聆听，流萤的语音朗读功能会将文字温柔地念给你听。它会逐句高亮当前朗读的内容，让声音与文字同步流淌。

---

## 三种声音来源

流萤支持三种 TTS（文本转语音）引擎，你可以在设置中选择最喜欢的一种：

### 1. 系统语音（默认）
使用设备自带的语音合成引擎。
- **优点**：无需配置，离线可用，零延迟
- **缺点**：音质因设备而异，部分设备音色较机械

### 2. Azure 语音服务
微软云端的高品质神经网络语音。
- **优点**：音色自然优美，情感丰富，支持多种风格（新闻、助手、聊天等）
- **缺点**：需要申请密钥，消耗云端字符额度（但有免费额度）
- **推荐**：追求最佳听书体验的用户

### 3. APP 原生 TTS（仅移动端）
如果安装在手机或平板上，流萤会优先调用系统级 TTS 引擎。
- **优点**：系统集成度高，响应快
- **缺点**：受限于设备厂商的语音包

## Azure 语音配置指南

### 为什么需要 Azure？

微软的 Azure 语音服务采用深度神经网络技术，合成的声音几可乱真。晓晓的温婉、云希的沉稳、云夏的少年感——每种音色都经过精心调校，足以陪伴你度过漫长的听书时光。

新用户每月有 **50 万字符的免费额度**，足够朗读 3-4 本长篇小说。

---

### 申请步骤

#### 第一步：注册 Azure 账号

1. 访问 [Azure 门户](https://portal.azure.com/)
2. 使用微软账号登录（或注册新账号）
3. 新用户自动获得 12 个月免费试用，包含 50 万字符/月的语音合成额度

#### 第二步：创建语音服务资源

1. 登录后点击左上角 **"创建资源"** (Create a resource)
2. 在搜索框输入 **"Speech"** 或 **"语音服务"**
3. 选择 **"Speech"** → 点击 **"创建"** (Create)
4. 填写以下配置：

| 配置项 | 建议填写 |
|--------|----------|
| 资源组 | 新建一个，如 "LuminaReader" |
| 区域 | **East Asia**（东亚）或离你最近的区域 |
| 名称 | 如 "lumina-tts" |
| 定价层 | **Free F0**（免费层，50万字符/月） |

5. 点击 **"审阅并创建"** → **"创建"**

等待约 1-2 分钟，资源部署完成。

#### 第三步：获取密钥

1. 进入刚创建的 Speech 资源页面
2. 左侧菜单点击 **"密钥和终结点"** (Keys and Endpoint)
3. 复制以下两项信息：
   - **KEY 1**（或 KEY 2，两个都可以使用）
   - **位置/区域**（如 `eastasia`）

> ⚠️ **重要**：密钥如同密码，请勿分享给他人或上传到公共仓库。

#### 第四步：配置到流萤

1. 在流萤阅读器中打开任意一本书
2. 点击顶部工具栏的 **齿轮图标** 进入设置
3. 找到 **"Azure 语音服务"** 区域
4. 将刚才复制的 **密钥** 粘贴到输入框
5. 选择你喜欢的**音色**和**风格**
6. 点击测试按钮确认可用

### 预加载音频缓存

在 Azure 设置卡片中，你可以调整两个参数来优化朗读体验：

| 参数 | 含义 | 建议值 |
|------|------|--------|
| **预加载句数** | 提前准备后面多少句的音频 | 5 句 |
| **缓存深度** | 总共缓存多少个预加载批次 | 5 倍 |

**计算方式**：总缓存句数 = 预加载句数 × 缓存深度（默认 25 句）

这意味着流萤会在后台悄悄为你准备好后面 5 句的音频，同时保留最近朗读过的 20 句。当你跳回之前的内容时，这些音频可以直接复用，无需再次向 Azure 请求——既节省了字符额度，又消除了等待的间隙。

### 常见问题

**Q：为什么有时候朗读会卡顿？**

A：首次朗读某句话时，流萤需要实时向 Azure 请求音频（约 0.5-2 秒）。从第二句开始，如果预加载已完成，就会流畅播放。建议保持"预加载音频"开关开启。

**Q：免费额度用完了怎么办？**

A：每月 1 日自动重置。如果当月额度用完，可以：
- 临时切换回系统语音
- 或升级为 Azure 的付费层（约 ¥4 元/千字符）

**Q：可以离线使用 Azure 语音吗？**

A：不可以。Azure 是云端服务，需要网络连接。但预加载的音频会缓存在本地，已缓存的内容可以重复播放而不再消耗额度。

**Q：为什么测试按钮没有声音？**

A：请检查：
1. 密钥和区域是否复制正确（注意空格）
2. 网络是否畅通（国内访问 Azure 可能需要梯子）
3. 浏览器是否阻止了自动播放（需要与页面交互后才能播放声音）
