## 二、我們收集的資訊

### 2.1 本地儲存的資料
以下資料僅儲存在您的裝置本地，我們不會上傳至任何伺服器：
- 您主動匯入的書籍文件
- 閱讀進度、書籤、標註、熱力圖資料
- 自訂字型檔案
- 應用程式設定偏好（主題、排版、TTS 配置等）

### 2.2 技術日誌資訊
在應用程式發生異常崩潰時，系統可能收集**脫敏的崩潰日誌**（不包含任何個人身份資訊），用於定位技術問題。此類資訊僅儲存於本地，不會自動上傳。

### 2.3 第三方服務互動
以下功能會涉及與第三方服務的資料互動：
- **語音朗讀（TTS）**：當您使用 Azure TTS 時，當前朗讀段落文字會透過加密通道傳輸至**微軟 Azure 語音服務**進行合成。我們僅在您主動開啟朗讀時傳輸必要文字，不做持久化儲存。

## 三、應用程式權限說明

本軟體所需的系統權限及用途如下：

- **INTERNET**：用於檢查應用程式更新、載入線上字型資源、以及 Azure TTS 語音合成服務。
- **WAKE_LOCK / FOREGROUND_SERVICE**：用於保障後台語音朗讀（TTS）不被系統中斷。
- **REQUEST_IGNORE_BATTERY_OPTIMIZATIONS**：用於向系統申請電池最佳化白名單，確保後台 TTS 穩定運行。
- **BLUETOOTH**：用於響應藍牙耳機播放/暫停按鍵事件。

**特別說明**：本軟體**不申請讀取/寫入外部儲存權限**。所有書籍匯入均透過 Android 系統檔案選擇器（SAF）由您主動選擇完成；匯出操作透過系統分享/儲存對話框完成。

## 五、我們如何保護您的資訊

1. **純本地儲存**：您的閱讀資料、書籍檔案、標註內容預設僅儲存在裝置本地資料庫或私有目錄中。
2. **應用程式沙盒隔離**：Android 系統的應用程式沙盒機制可防止其他應用程式存取本軟體資料。
3. **加密備份**：您匯出的 .lmn 備份檔案採用 AES-256-GCM 加密，需密碼才能解密。
4. **無帳號體系**：本軟體不註冊、不收集任何使用者帳號或身份資訊。

## 六、您的權利

您對自己的資料擁有以下權利：

1. **查閱與匯出**：透過「設定 → 資料管理 → 匯出」，您可隨時將書籍和配置資料匯出為 .lmn 備份檔案。
2. **刪除**：透過「設定 → 資料管理 → 清除全部資料」，可一鍵刪除所有本地資料；解除安裝應用程式也將徹底清除所有殘留資料。
3. **撤回同意**：您可隨時在應用程式設定中關閉 TTS 功能，停止向 Azure 傳輸文字資料。
