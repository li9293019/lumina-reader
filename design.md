## Azure TTS 缓存设置区域设计

### 布局结构
```
┌─────────────────────────────────────┐
│  [开关]  预加载音频                    │  ← 标题行，开关在左
│          提前合成后面几句，朗读更流畅    │  ← 副标题说明
├─────────────────────────────────────┤
│  预加载句数  [────●────]  5句         │  ← 滑块行
├─────────────────────────────────────┤
│  ┌─────────┬─────────┬─────────┐   │
│  │  80%    │   12    │  2.4k   │   │  ← 统计卡片（横向3列）
│  │ 命中率  │ 已缓存  │节省字符 │   │
│  └─────────┴─────────┴─────────┘   │
├─────────────────────────────────────┤
│                    [  清空缓存  ]     │  ← 右下角按钮
└─────────────────────────────────────┘
```

### 样式规范

**容器**
- 背景：var(--bg-secondary) 带轻微透明
- 边框：1px solid var(--border-color)
- 圆角：12px
- 内边距：16px
- 上边距：20px（与上方 STYLE 区域分隔）

**开关行**
- 开关使用现有 .azure-toggle-switch 样式
- 标题：font-weight: 500, font-size: 14px
- 副标题：font-size: 12px, color: var(--text-secondary)

**滑块行**
- 标签：font-size: 13px
- 滑块：height: 4px, 圆角 2px
- 数值：font-weight: 500, color: var(--accent-color)

**统计卡片**
- 3列等宽网格
- 卡片背景：var(--bg-primary)
- 数值：font-size: 18px, font-weight: 600, color: var(--accent-color)
- 标签：font-size: 11px, color: var(--text-secondary), 大写

**清空按钮**
- 使用胶囊样式
- 位置：右对齐
- 颜色：默认灰色，hover 变红色
