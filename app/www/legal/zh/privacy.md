## 二、我们收集的信息

### 2.1 本地存储的数据
以下数据仅保存在您的设备本地，我们不会上传至任何服务器：
- 您主动导入的书籍文档
- 阅读进度、书签、批注、热力图数据
- 自定义字体文件
- 应用设置偏好（主题、排版、TTS 配置等）

### 2.2 技术日志信息
在应用发生异常崩溃时，系统可能收集**脱敏的崩溃日志**（不包含任何个人身份信息），用于定位技术问题。此类信息仅存储于本地，不会自动上传。

### 2.3 第三方服务交互
以下功能会涉及与第三方服务的数据交互：
- **语音朗读（TTS）**：当您使用 Azure TTS 时，当前朗读段落文本会通过加密通道传输至**微软 Azure 语音服务**进行合成。我们仅在您主动开启朗读时传输必要文本，不做持久化存储。

## 三、应用权限说明

本软件所需的系统权限及用途如下：

- **INTERNET**：用于检查应用更新、加载在线字体资源、以及 Azure TTS 语音合成服务。
- **WAKE_LOCK / FOREGROUND_SERVICE**：用于保障后台语音朗读（TTS）不被系统中断。
- **REQUEST_IGNORE_BATTERY_OPTIMIZATIONS**：用于向系统申请电池优化白名单，确保后台 TTS 稳定运行。
- **BLUETOOTH**：用于响应蓝牙耳机播放/暂停按键事件。

**特别说明**：本软件**不申请读取/写入外部存储权限**。所有书籍导入均通过 Android 系统文件选择器（SAF）由您主动选择完成；导出操作通过系统分享/保存对话框完成。

## 五、我们如何保护您的信息

1. **纯本地存储**：您的阅读数据、书籍文件、批注内容默认仅保存在设备本地数据库或私有目录中。
2. **应用沙盒隔离**：Android 系统的应用沙盒机制可防止其他应用访问本软件数据。
3. **加密备份**：您导出的 .lmn 备份文件采用 AES-256-GCM 加密，需密码才能解密。
4. **无账号体系**：本软件不注册、不收集任何用户账号或身份信息。

## 六、您的权利

您对自己的数据拥有以下权利：

1. **查阅与导出**：通过「设置 → 数据管理 → 导出」，您可随时将书籍和配置数据导出为 .lmn 备份文件。
2. **删除**：通过「设置 → 数据管理 → 清除全部数据」，可一键删除所有本地数据；卸载应用也将彻底清除所有残留数据。
3. **撤回同意**：您可随时在应用设置中关闭 TTS 功能，停止向 Azure 传输文本数据。
